## Useful stuff

## Conventions
* filenames for web 
  * Lowercase. e.g. index.html or my_script.css
  * Always start with a letter 
* Variables
  * camelCase. e.g. myParagraph; or showImage();
  * const and let are used, not var!
* Classes
  * hyphens. e.g. company-info-picture

## Wordlist
* ; = semicolon
* : = colon
* . = dot
* _ = underscore
* \- = hyphen
* <> = brackets/brace
* () = parenthesis
* & = ampersand
* " " = quotes
* {} = curly brace
* \* = asterisk
* ! = exclamation mark
* / = slash
* \ = backslash
